## 1. CONTEXT
## ─────────────────────────────────────────────

The current POST /api/generate/code endpoint generates Phaser.js code
regardless of the target engine. This is wrong.

Redesign it to generate engine-specific scripts based on project.target_engine:
  unity   → C# MonoBehaviour scripts
  unreal  → C++ header + implementation
  godot   → GDScript files
  phaser  → JavaScript (keep existing behavior as fallback)

The primary use case for V57 Studio is Unity. Focus quality on that.

```javascript
// code.prompt.js

function buildCodePrompt(targetEngine, gameContext) {
  switch (targetEngine) {
    case 'unity':   return buildUnityPrompt(gameContext)
    case 'unreal':  return buildUnrealPrompt(gameContext)
    case 'godot':   return buildGodotPrompt(gameContext)
    case 'phaser':
    default:        return buildPhaserPrompt(gameContext)
  }
}

function buildUnityPrompt(ctx) {
  return `You are an expert Unity game developer (C#, Unity 2022.3 LTS).

Generate 4 complete, compilable C# Unity scripts for this game based on the GDD.
Name all classes using the actual game name and characters from the GDD.

GAME CONTEXT:
${JSON.stringify(ctx, null, 2)}

SCRIPTS TO GENERATE:

1. [GameName]PlayerController.cs
   - MonoBehaviour attached to the Player GameObject
   - Rigidbody2D or CharacterController based movement
   - Left/right movement, jumping with ground detection
   - Implement the FIRST core mechanic from GDD mechanics array
   - Health system with the number of lives from GDD
   - SerializeField for all tunable values (speed, jumpForce, etc.)
   - XML documentation comments on all public methods
   - TODO comments where developer must add animations, VFX, SFX

2. [GameName]EnemyController.cs
   - Base enemy class + specific behavior per enemy type from GDD
   - Each enemy type from GDD characters array gets its own behavior
   - Patrol AI: move between waypoints or bounce off walls
   - Player detection with configurable detection radius
   - Health and damage system
   - OnDeath event for spawning effects or drops

3. [GameName]GameManager.cs
   - Singleton pattern (Instance property)
   - Game state enum: MainMenu, Playing, Paused, GameOver, Victory
   - Health/lives tracking matching GDD
   - Level progression system
   - Events: OnHealthChanged, OnGameStateChanged, OnLevelComplete
   - SceneManager integration for level transitions

4. ForgeManifest.cs
   - Static class, auto-generated by Forge — mark with comment DO NOT EDIT
   - Constants from GDD: game name, genre, version, target engine
   - String arrays: mechanics names, character names, level names
   - Build metadata: generated date, Forge version

UNITY C# RULES:
- All scripts must compile without errors in Unity 2022.3 LTS
- Use UnityEngine namespace correctly
- SerializeField for inspector-exposed private fields
- Use [Header("Section Name")] to organize inspector fields
- Use Awake() for self-initialization, Start() for cross-references
- Use FixedUpdate() for physics, Update() for input
- No hardcoded magic numbers — use named constants or SerializeField
- Add [RequireComponent] attributes where appropriate
- Mark TODO items as: // TODO: [description of what developer must add]
- Use proper C# conventions: PascalCase for classes/methods, camelCase for fields

RETURN FORMAT:
Return ONLY valid JSON. No markdown, no backticks outside the JSON.
All C# code strings must have newlines as \\n and quotes escaped as \\".

{
  "engine": "unity",
  "files": [
    {
      "filename": "[GameName]PlayerController.cs",
      "description": "Player movement and core mechanic controller",
      "content": "using UnityEngine;\\n\\npublic class..."
    },
    {
      "filename": "[GameName]EnemyController.cs", 
      "description": "Enemy AI and behavior controller",
      "content": "using UnityEngine;\\n\\npublic class..."
    },
    {
      "filename": "[GameName]GameManager.cs",
      "description": "Game state and progression manager",
      "content": "using UnityEngine;\\n\\npublic class..."
    },
    {
      "filename": "ForgeManifest.cs",
      "description": "Auto-generated GDD constants — do not edit",
      "content": "// AUTO-GENERATED BY FORGE\\n\\npublic static class..."
    }
  ],
  "architecture_md": "# [GameName] — Unity Architecture\\n\\n..."
}`
}

function buildPhaserPrompt(ctx) {
  return `You are an expert Phaser.js 3 game developer.

Generate a COMPLETE, WORKING Phaser.js 3 platformer game based on the GDD provided.

MANDATORY REQUIREMENTS:
1. Use Phaser.Scene classes (NOT function-based scenes)
2. Create PreloadScene, GameScene, and GameOverScene classes
3. Player must have: left/right movement, jumping, gravity
4. Include at least 2 enemy instances with patrol AI
5. Include platforms using staticGroup with colored rectangles
6. Implement the FIRST core mechanic from the GDD
7. Player uses keyboard: arrow keys for movement, SHIFT or E for special mechanic
8. Camera must follow the player
9. Include health system (3 lives) and game over logic
10. Name ALL classes using the actual game characters from the GDD

CRITICAL - PLACEHOLDER GRAPHICS:
- NEVER use this.physics.add.sprite() with a texture key
- NEVER load images in preload()
- NEVER call .create() on a staticGroup with a texture key
- For player: this.add.rectangle() then this.physics.add.existing(rect)
- For static platform: this.add.rectangle() then this.physics.add.existing(rect, true)
- For enemy: this.add.rectangle() then this.physics.add.existing(rect)

Return ONLY valid JSON:
{
  "engine": "phaser",
  "files": [
    { "filename": "game.js", "description": "Complete Phaser.js 3 game", "content": "..." }
  ],
  "architecture_md": "..."
}`
}
```

```javascript
const { buildCodePrompt } = require('../prompts/code.prompt')

router.post('/code', async (req, res, next) => {
  try {
    const { project_id } = req.body

    if (!project_id) {
      return res.status(400).json({
        success: false,
        error: 'project_id is required',
        code: 'VALIDATION_ERROR'
      })
    }

    const { data: project, error } = await db()
      .from('projects').select('*').eq('id', project_id).single()
    if (error || !project) {
      return res.status(404).json({
        success: false, error: 'Project not found', code: 'NOT_FOUND'
      })
    }

    const targetEngine = project.target_engine || 'unity'

    // Build compact game context to avoid token limits
    const gameContext = {
      name: project.concept.project?.name,
      genre: project.concept.project?.genre,
      core_loop: project.concept.project?.core_loop,
      tone: project.concept.project?.tone,
      target_engine: targetEngine,
      mechanics: project.concept.mechanics?.slice(0, 3).map(m => ({
        name: m.name,
        type: m.type,
        description: m.description
      })),
      characters: project.concept.characters?.map(c => ({
        name: c.name,
        role: c.role,
        abilities: c.abilities?.slice(0, 3),
        description: c.description?.slice(0, 150)
      })),
      levels: project.concept.levels?.slice(0, 3).map(l => ({
        name: l.name,
        difficulty: l.difficulty,
        environment: l.environment
      })),
      art_direction: {
        style: project.concept.art_direction?.style,
        resolution: project.concept.art_direction?.resolution
      }
    }

    const systemPrompt = buildCodePrompt(targetEngine, gameContext)

    let result
    try {
      result = await callLLM(
        systemPrompt,
        `Generate the ${targetEngine} scripts for: ${gameContext.name}`,
        { step: 'step_4_code', maxOutputTokens: 8000 }
      )
    } catch (err) {
      const isRateLimit = err.status === 429
      return res.status(502).json({
        success: false,
        error: isRateLimit
          ? 'Rate limit reached. Try again later or switch models.'
          : 'LLM API call failed',
        code: isRateLimit ? 'RATE_LIMIT' : 'LLM_ERROR',
        retry_after_ms: err.retry_after_ms || null,
        ...(process.env.NODE_ENV === 'development' && {
          details: { message: err.message }
        })
      })
    }

    // Validate response has files array
    const { files, architecture_md, engine } = result.data
    if (!files || !Array.isArray(files)) {
      return res.status(502).json({
        success: false,
        error: 'LLM returned invalid script structure',
        code: 'INVALID_RESPONSE'
      })
    }

    // Save files to disk
    ensureProjectDir(project_id)
    const codePath = require('path').join(STORAGE_BASE, 'projects', project_id, 'code')

    const savedFiles = []
    for (const file of files) {
      const filePath = require('path').join(codePath, file.filename)
      require('fs').writeFileSync(filePath, file.content || '')
      savedFiles.push({
        filename: file.filename,
        description: file.description,
        url: getAssetUrl(project_id, `code/${file.filename}`),
        size_bytes: Buffer.byteLength(file.content || '', 'utf8')
      })
    }

    // Save architecture.md
    if (architecture_md) {
      require('fs').writeFileSync(
        require('path').join(codePath, 'architecture.md'),
        architecture_md
      )
    }

    res.status(200).json({
      success: true,
      engine: targetEngine,
      files: savedFiles,
      architecture_md: architecture_md || '',
      meta: result.meta
    })
  } catch (err) {
    next(err)
  }
})
```

```javascript
router.post('/:project_id/approve-step4', async (req, res, next) => {
  try {
    const { project_id } = req.params
    const { files, architecture_md, engine } = req.body

    // Support both old format { code: { game_js, architecture_md } }
    // and new format { files: [...], architecture_md, engine }
    const isNewFormat = Array.isArray(files)
    const isOldFormat = req.body.code && req.body.code.game_js

    if (!isNewFormat && !isOldFormat) {
      return res.status(400).json({
        success: false,
        error: 'files array or code.game_js is required',
        code: 'VALIDATION_ERROR'
      })
    }

    const { data: project, error: pErr } = await db()
      .from('projects').select('id').eq('id', project_id).single()
    if (pErr || !project) {
      return res.status(404).json({
        success: false, error: 'Project not found', code: 'NOT_FOUND'
      })
    }

    const { data: job } = await db()
      .from('generation_jobs')
      .insert({
        project_id,
        triggered_by: TEST_MEMBER_ID,
        status: 'approved',
        progress: 100,
        current_step: 'step_4_code',
        input_prompt: `Code for project ${project_id}`,
        started_at: new Date().toISOString(),
        completed_at: new Date().toISOString()
      })
      .select().single()

    const createdAssets = []

    if (isNewFormat) {
      // New format: array of files
      for (const file of files) {
        const { data: asset } = await db()
          .from('assets')
          .insert({
            project_id,
            job_id: job?.id,
            name: file.filename,
            type: 'code',
            discipline: 'code',
            review_status: 'approved',
            reviewed_by: TEST_MEMBER_ID,
            reviewed_at: new Date().toISOString()
          })
          .select().single()

        if (asset) {
          await db().from('asset_versions').insert({
            asset_id: asset.id,
            version_number: 1,
            source: 'ai_generated',
            storage_url: getAssetUrl(project_id, `code/${file.filename}`),
            storage_bucket: 'local',
            model_used: `${engine || 'unity'}-generated`,
            uploaded_by: TEST_MEMBER_ID,
            is_current: true,
            metadata: {
              engine: engine || 'unity',
              description: file.description,
              size_bytes: file.size_bytes
            }
          })
          createdAssets.push(asset)
        }
      }

      // Save architecture.md as asset too
      if (architecture_md) {
        const { data: archAsset } = await db()
          .from('assets')
          .insert({
            project_id,
            job_id: job?.id,
            name: 'architecture.md',
            type: 'code',
            discipline: 'code',
            review_status: 'approved',
            reviewed_by: TEST_MEMBER_ID,
            reviewed_at: new Date().toISOString()
          })
          .select().single()

        if (archAsset) {
          await db().from('asset_versions').insert({
            asset_id: archAsset.id,
            version_number: 1,
            source: 'ai_generated',
            storage_url: getAssetUrl(project_id, 'code/architecture.md'),
            storage_bucket: 'local',
            model_used: 'ai_generated',
            uploaded_by: TEST_MEMBER_ID,
            is_current: true
          })
          createdAssets.push(archAsset)
        }
      }
    } else {
      // Old format backward compatibility
      const { game_js, architecture_md: old_arch } = req.body.code
      for (const [name, storageUrl] of [
        ['game.js', getAssetUrl(project_id, 'code/game.js')],
        ['architecture.md', getAssetUrl(project_id, 'code/architecture.md')]
      ]) {
        const { data: asset } = await db()
          .from('assets')
          .insert({
            project_id, job_id: job?.id, name,
            type: 'code', discipline: 'code',
            review_status: 'approved',
            reviewed_by: TEST_MEMBER_ID,
            reviewed_at: new Date().toISOString()
          })
          .select().single()

        if (asset) {
          await db().from('asset_versions').insert({
            asset_id: asset.id, version_number: 1,
            source: 'ai_generated', storage_url: storageUrl,
            storage_bucket: 'local', model_used: 'phaser-generated',
            uploaded_by: TEST_MEMBER_ID, is_current: true
          })
          createdAssets.push(asset)
        }
      }
    }

    res.status(201).json({
      success: true, job_id: job?.id, assets: createdAssets
    })
  } catch (err) {
    next(err)
  }
})
```
